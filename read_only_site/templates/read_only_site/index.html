Welcome to the Hard Drop League!

How do I register?

Go to the Hard Drop Discord (https://discord.gg/harddrop), and type !register in the #tournament channel.

How do I submit my score?

The loser of the match has to type !winner @mention, with @mention mentionning the winner.
